#  Amazon Sentiment Analysis

This project is a **Machine Learning + NLP based Sentiment Analysis system** that classifies Amazon product reviews into **Positive** or **Negative** sentiments.

---

# Features
- Analyze customer reviews automatically
- Classify sentiment (Positive / Negative)
- Clean and preprocess text using NLP
- High accuracy model (~90%)

---

# Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- NLP (Text Preprocessing, TF-IDF)

---

##  Model Performance
- Accuracy: **~90%**
- Model: Machine Learning Classifier
